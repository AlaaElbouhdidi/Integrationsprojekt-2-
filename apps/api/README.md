# Integrationsprojekt 2 - Backend

View function deployment at <https://europe-west1-integrationsprojekt2.cloudfunctions.net/api>

## Installation

```bash
npm install
```

## Running the app

```bash
# development
npm run start

# watch mode
npm run start:dev

# production mode
npm run start:prod
```

## Test

```bash
# unit tests
npm run test

# e2e tests
npm run test:e2e

# test coverage
npm run test:coverage
```

## Deployment

```bash
# Deploy with firebase functions
# Note: This might take 3-5 minutes!

npm run deploy
```
